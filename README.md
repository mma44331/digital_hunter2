איך להריץ את הפרויקט:

וודא ש-Docker Desktop פתוח ועובד.

פתח טרמינל בתיקיית השורש של הפרויקט (איפה שנמצא הקובץ docker-compose.yaml).

הרץ את הפקודה: docker-compose up -d --build

המתן לסיום הבנייה של ה-Images ועליית הקונטיינרים.

מה ניתן לעשות בפרויקט:

המערכת משמשת כמעבד נתונים מודיעיניים (Digital Hunter). היא צורכת אותות מחיישנים שונים דרך Kafka, מבצעת וולידציה של השדות ומחשבת מרחקי תנועה של מטרות ניידות (Haversine). המידע נשמר ב-MongoDB וניתן לתשאל אותו דרך ממשק Swagger בכתובת http://localhost:8000/docs.

דרך הממשק ניתן להפיק תובנות כמו:

איתור מטרות בעדיפות גבוהה שביצעו תזוזה משמעותית בשטח.

ניתוח ודירוג של מקורות המידע (Signal Types) מהנפוץ ביותר לנדיר ביותר.

צפייה בכלל המטרות שנקלטו במערכת.
